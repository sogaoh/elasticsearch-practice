# elasticsearch-practice

## これは何？
個人的に Elasticsearch を練習する意図の環境起動用ファイル一式です

## 起動方法
```
git clone https://github.com/sogaoh/elasticsearch-practice.git
cd elasticsearch-practice

docker-compose up -d
```

## Applications
- Elasticsearch
    - http://localhost:9200/
- Kibana 
    - http://localhost:5601/

## Plugins
- Elasticsearch
    - kuromoji
